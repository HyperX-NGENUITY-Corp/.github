## 🧠 Overview

**HyperX NGENUITY** is a desktop application for configuring compatible HyperX gaming peripherals.

It lets users customize RGB lighting, create macros, fine-tune device performance, and manage individual profiles.

The software also supports firmware updates and onboard memory configuration through a streamlined interface.

Perfect for gamers, streamers, esports players, and Windows users with HyperX devices. :contentReference[oaicite:0]{index=0}

## 🚀 Key Features

| Feature | Description |
|----------|-------------|
| 🌈 **RGB Lighting Control** | Personalize lighting effects with custom colors, zones, and animations. |
| ⌨️ **Macro Programming** | Assign shortcuts and multi-action macros to supported buttons and keys. |
| 🎯 **Performance Tuning** | Adjust DPI, sensitivity, polling rate, and device behavior. |
| 🎮 **Custom Profiles** | Create and automatically switch between application-specific profiles. |
| 💾 **Onboard Memory Support** | Save settings directly to compatible HyperX peripherals. |
| 🎧 **Audio Configuration** | Customize supported headset audio, microphone, and playback options. |
| 🔄 **Firmware Updates** | Install the latest firmware for improved compatibility and performance. |
| 🖥️ **Unified Dashboard** | Control all supported HyperX devices from one application. |
